# Application Validation Evidence

Validation records for the demonstrator:

- `01_app_validation_manifest.csv` indexes validation modes and logs.
- `02_app_runtime_summary.csv` records runtime measurements.
- `03_app_failure_log.csv` records execution failures.
- `04_app_readiness_status.json` provides the machine-readable status.

The compact repository retains aggregate validation records rather than bulk
per-run images and logs.
