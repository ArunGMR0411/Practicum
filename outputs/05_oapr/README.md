# Objective-Aware Privacy Router Evidence

Canonical RQ4 routing evidence. Top-level files record selection criteria,
routing and materialisation logs, runtime, failures, per-frame metrics,
fixed-method comparisons, ablations, and oracle analysis.

- `decision_framework/` contains progressive stage gates and deployment scores.
- `routing_support/` contains supporting routing analyses.
- `cross_view_analysis/` contains the bounded paired-view evaluation.

The exact report claims and source tables are mapped in `../09_traceability/`.
