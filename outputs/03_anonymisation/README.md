# Face Anonymisation Evidence

Canonical RQ2 evidence on the locked 500-image protocol. It covers
deterministic methods, advanced-method execution and failures, privacy and
utility metrics, runtime, visual review, eligibility gates, and final policy
selection.

Start with `01_all_methods_comparison.csv`, then use the numbered method and
policy directories for the underlying measurements. Missing outputs and
non-promoted methods remain visible in their failure or eligibility records.
