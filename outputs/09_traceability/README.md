# Evidence Traceability

Indexes connecting thesis claims, research questions, tables, and retained
evidence:

- `01_evidence_index.csv`: evidence inventory and interpretation.
- `02_claim_to_evidence_map.csv`: claim-level sources.
- `03_table_to_source_map.csv`: report-table sources.
- `08_rq_to_evidence_map.csv`: research-question sources.
- `09_missing_excluded_file_ledger.csv`: excluded or unavailable artefacts.
- `10_final_stage_comparison.csv`: compact final-stage comparison.

Integrity records preserve model-weight and raw-dataset checks without
redistributing restricted assets.
