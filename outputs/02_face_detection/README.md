# Face Detection Evidence

Canonical RQ1 results for the reviewed baseline and egocentric-stress
500-image protocols. The directory contains detector comparisons, condition
profiles, hardening experiments, candidate-box telemetry, runtime-source
validation, and final deployment-selection evidence.

`03_detector_two_protocol_comparison.csv` is the compact cross-protocol result.
`13_anonymisation_protocol_face_boxes.csv` is the reviewed-box handoff to RQ2.
Detailed provenance is indexed in `../09_traceability/`.
